My recommendation for Afterglow's signature look:

Buttons: Cut-corner (sharp 0dp + 12dp clipped top-left & bottom-right) with optional inline D-pad key hint ·
EPG cells: Beveled corners (top-left + bottom-right clipped 10dp) · NOW cells get accent fill + stripe + top border + corner tag ·
Text fields: Already-shipped accent-stripe (2dp corners + 4dp leading stripe, expands to 6dp on focus + accent-faint fill) ·
Channel rows: Accent stripe + 2dp now-progress bar pinned to bottom edge ·
Pills: Striped (mini 3dp accent stripe per chip) · LIVE / REC use solid + blink animation ·
Focus: Stripe + fill (current shipping behaviour — covers rows, fields, cards, EPG cells, pills) ·
Progress: Hairline 2dp on every row · pill style on the player scrubber.

This combo gives Afterglow three signatures across every surface: accent stripe, beveled corners (on cells & primary buttons), and the 2-dp hairline progress on rows. No other IPTV app does any of those, let alone all three. Tell me which of these shapes feels right and I'll start wiring them in.
